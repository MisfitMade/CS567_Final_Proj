Technical updates are updates which generally deal with improvements to servers, or maintenance. The Old School RuneScape development team have typically favour website updates in most cases, however on occasion use this category.

Chronological list of updates[edit | edit source]

Technical updates are listed in reverse chronological order. For an alphabetical list, see Category:Technical updates.
